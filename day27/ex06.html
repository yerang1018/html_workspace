a tag(link)

문서에서 다른 문서, 다른 파일, 다른 위치로 링크를 만드는 태그

문서 끝으로

  • a태그는 글자(및 그림)에 링크를 부여
  • a태그는 줄변경을 발생시키지 않는다
  • a태그의 대상은 html 문서 혹은 웹 주소, 파일이 될 수 있다
  • a태그의 대상이 현재 문서내부의 다른 요소가 될 수 있다
  • href : Http Reference , 링크
  • target : 링크가 열릴 창,
  • a태그의 대상이 브라우저가
